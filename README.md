## ğŸ“¦ Deployment

### Deploy to Vercel

1. Fork this repository
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Import your forked repository
5. Vercel will auto-detect the configuration
6. Click "Deploy"

### Environment Setup

You'll need to set up your own Supabase project:

1. Create a project at [Supabase](https://supabase.com)
2. Run the SQL schema (see `SUPABASE_SETUP.md` if available)
3. Update `supabase-client.js` with your:
   - Project URL
   - Anon/Public Key
